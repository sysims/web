# Company Portal Frontend

Web portal perusahaan berbasis React (Vite) yang mencakup fitur Absensi, Cuti, dan Info Perusahaan.

## Teknologi yang Digunakan
- React + Vite
- Tabler (UI Framework)
- ESLint + Prettier

## Cara Menjalankan Proyek
1. Pastikan Node.js sudah terinstal.
2. Jalankan `npm install` untuk mengunduh dependencies.
3. Jalankan `npm run dev` untuk menyalakan server lokal.
